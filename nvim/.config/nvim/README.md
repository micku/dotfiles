## Add Python support

```
pyenv install -l # List and choose versions

pyenv install 2.7.18
pyenv virtualenv 2.7.18 neovim2
pyenv activate neovim3
pip install neovim
source deactivate

pyenv install 3.9.1
pyenv virtualenv 3.9.1 neovim3
pyenv activate neovim3
pip install neovim
source deactivate
```
